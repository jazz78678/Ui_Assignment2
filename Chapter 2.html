Chapter 2

Website should be according to user's need. User should be satisfied from website's design. Every point should clearly defined. Every button should easily clickable. Website should responsive. User should easy to choose their choice. Website should have only one goal to satisfy the user. Home button should easily approachable because it is the important button. User should know about the website from first look. Search box on the website should be simple and clear to the user. Website should have less pop-ups. Website should have interactive user interface. Content should not be underlined. Reference Should almost match the page, otherwise user will confuse.
